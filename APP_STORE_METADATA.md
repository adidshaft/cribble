## Name

Cribble: Markdown Knowledge Base Manager

## Subtitle

Native Markdown reading for local knowledge folders.

## Promotional Text

Turn plain Markdown folders into a calm, connected Mac-native reading space.

## Description

Cribble is a native macOS Markdown knowledge base manager for people who keep
their notes, docs, research, and project plans in plain `.md` files.

Open one or more local folders, browse them in a familiar sidebar, and read
Markdown with polished typography, wiki links, relative images, code blocks,
tables, task lists, and math support. Cribble keeps editing in your existing
tools while making reading and navigation feel fast, focused, and native to the
Mac.

Cribble is local-first. Your files remain ordinary Markdown on disk, and the app
does not require an account or cloud sync.

Core features:

- Browse multiple local Markdown folders.
- Read rendered Markdown with clean Mac-native typography.
- Follow wiki links like `[[Home]]`, `[[Note#Heading]]`, and `[[Note|Label]]`.
- Open notes in your preferred editor when you want to make changes.
- Show linked files as compact cards for easier navigation.
- Use preview-first AI linking with local Codex or Claude command-line tools.

Cribble is intentionally a reader and connector, not another writing surface.
It is built for users who already trust their Markdown files and want a better
way to move through them.

## Review Notes

Cribble opens user-selected local folders containing Markdown files. It stores
security-scoped bookmarks so those folders can be reopened between launches.

The AI linking feature is optional and only runs if the user already has a
supported local command-line tool installed. AI suggestions are shown as a
native patch preview before any file is changed.

The app does not require an account, does not include an in-app paywall, and
does not include in-app purchases.

## Privacy Notes

- No account is required.
- User documents stay on the local Mac unless the user separately chooses to use
  an external tool.
- No analytics SDK is included.
- Local folder access is user-selected.
