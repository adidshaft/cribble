# Cribble Privacy Policy

Cribble is a local-first macOS Markdown reader and knowledge base manager.

Cribble does not require an account, does not include advertising or third-party analytics, and does not collect personal data.

The app opens Markdown folders selected by the user. Those files stay on the user's Mac unless the user stores them in a separately configured cloud folder, such as iCloud Drive, Dropbox, or Google Drive. Cribble stores local app preferences and security-scoped bookmarks so selected folders can be reopened between launches.

Cribble includes optional AI-assisted linking features. These features only run when the user has installed and configured a supported local command-line AI tool. Cribble shows suggested file changes for review before anything is written to disk.

Cribble does not sell user data, track users across apps or websites, or transmit user documents to a Cribble-operated server.

For privacy questions, open an issue at:

https://github.com/adidshaft/cribble/issues
