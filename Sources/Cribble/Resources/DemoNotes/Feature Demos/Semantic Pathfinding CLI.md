# Semantic Pathfinding + CLI

Pathfinder connects two notes in two ways:

1. It looks for the shortest wiki-link path already present in the library.
2. It asks the local semantic index for conceptual stepping stones between the notes.

To try it, drag [[Mermaid Math Zoom Overlays]] onto this note in the sidebar. The existing wiki path should pass through [[Reading Trails]] and [[Local Semantic Search]], while the conceptual bridge can point at semantically nearby notes.

The explanation menu can call a local Claude or Codex CLI in read-only mode to describe why two notes relate. This is useful when a path exists but the connection is not obvious from titles alone.
